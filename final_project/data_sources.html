Data Download

The dataset can be downloaded here.

It can also be viewed as an interactive map here. To view different variables:

  • Click the “layers” tab on the upper left corner
  • Select the “ES-202 Final Shapefile” layer
  • Select the “styles” tab on the upper right corner
  • Choose 1-2 “atttributes” using the shapefile variable names above
  • Pick a style to map the variables (I like “relationship” for 2 variables, and “counts and amounts” for 1 variable)
